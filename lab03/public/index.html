Lab03 Answers:

  • 3.1
    • Node is a runtime for JavaScript whereas Express is a web framework that makes serving HTML using JS easier.
    • The package.json file tells npm about dependencies in the code, and gives basic information about the app
      such as author, name, version and license information.
  • 3.2
    • The files served up by this website are static in the sense that they are loaded once and displayed. They aren't
      actively changing like a dynamic webpage would be.
    • The HelloWorld text sent to the console/webpage does not continue to be sent to the client. Once the server sends this
      there is no need to continue sending it. Its already been rendered on the screen.
